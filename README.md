## Markup Rule
- 전체를 .wrap으로 감싸고 header + section*3으로 나눈다. 
- h1은 사이트의 제목을 표시하고, h2는 각 section 안에 위치한다.
- 반응형으로 작업하되, pc버전에서만 FullPage로 작업한다.
